## Development install

For a **development install**:

1. Clone the repository:
   ```bash
   $ git clone https://github.com/jupyterhub/dockerspawner
   ```
2. Install from source:
   ```bash
   $ cd dockerspawner
   $ python3 -m pip install -r dev-requirements.txt -e .
   ```
